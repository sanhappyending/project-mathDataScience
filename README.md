# Project Math for Data Science

Proyek implementasi probabilitas, statistika, distribusi data, dan A/B Testing menggunakan Python untuk kebutuhan Data Science dan Machine Learning.

## 📌 Deskripsi Project

Project ini merupakan submission akhir dari kelas **Belajar Matematika untuk Data Science**.

Pada project ini dilakukan implementasi berbagai konsep matematika, probabilitas, dan statistika menggunakan Python untuk menyelesaikan studi kasus data science, meliputi:

* Distribusi statistik
* Probabilitas
* Naive Bayes
* A/B Testing
* Sistem rekomendasi
* Analisis statistik inferensial

Project bertujuan untuk memperkuat pemahaman terhadap penerapan matematika dalam workflow Data Science dan Artificial Intelligence.

---

## 🎯 Objectives

* Mengimplementasikan konsep probabilitas menggunakan Python
* Memahami distribusi statistik pada data
* Melakukan analisis A/B Testing
* Menggunakan statistik inferensial untuk pengambilan keputusan
* Menerapkan konsep matematika pada algoritma AI
